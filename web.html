AI in Education Quiz

Description: This project is an interactive quiz website that explores the theme of AI in Education, designed to inform users about the role and impact of artificial intelligence within educational settings. The quiz is user-friendly and includes a unique "Cheat Mode" feature, allowing users to reveal correct answers with detailed explanations if needed. Visual elements like an animated cat add a fun, interactive touch, making the quiz both engaging and informative.

Creation Process: The quiz was created using Notepad for code editing and writing in HTML, CSS, and JavaScript. HTML and CSS were used to structure and style the quiz interface, while JavaScript enabled the interactive quiz functionality. The project was tested locally before being published on GitHub Pages, which provides free static web hosting and makes the site accessible to users.

Tools:

  • Notepad: Used as the primary code editor to write HTML, CSS, and JavaScript.
  • HTML/CSS: For structuring and styling the quiz interface.
  • JavaScript: For implementing quiz logic, managing user feedback, and adding interactive animations.
  • GitHub Pages: Used to host the project and share it online.

AI in Education Drag-and-Drop Quiz

This project transforms a traditional drag-and-drop quiz into an interactive and visually engaging experience, with the theme focused on “AI in Education.” Below are the details of the updates I made, including the theme modification, CSS improvements, and enhanced user interactivity.

1. Theme Change to “AI in Education”

Concept Update: I updated it to focus on “AI in Education,” a modern and relevant topic in educational technology. This theme helps users explore and understand how AI is applied in educational contexts.

Categories and Concepts:

  • Personalized Learning: Concepts like “Adaptive Learning” and “Personalized Study Path” help students receive customized educational experiences.
  • Automated Assessment: Includes concepts such as “Automated Grading” and “AI-based Feedback,” highlighting AI’s role in grading and feedback automation.
  • Teaching Assistance: Concepts like “Virtual Tutor” and “Teacher Support Tools” illustrate how AI can support educators in their teaching tasks.

2. CSS Styling Enhancements

I made the following updates to the CSS to improve the quiz's visual appeal and usability:

  • Soft Color Scheme and Shadows: Introduced a professional and modern color palette with softer tones that enhance readability and reduce visual strain.
  • Rounded Corners: Added border-radius to draggable items and drop zones, creating a smoother, approachable interface.
  • Responsive Layout Adjustments: Optimized the layout to ensure it is accessible on both desktop and mobile devices.

3. Interactivity Improvements

To make the quiz experience more engaging and intuitive, I added:

  • Hover Effects and Enlargement: Draggable items slightly enlarge on hover, giving users a clear indication of interactivity.
  • Color Change on Drop Zones: Drop zones highlight when hovered over, making the interface more intuitive.
  • Scroll Functionality for Overflowing Lists: Added a scroll bar to keep the interface clean while accommodating more content.
